
>容器说明

在yum服务器的容器里面，最重要的一个目录就是那个“repodata”，该目录是分析RPM软件后所产生的软件依赖数据放置处。因此，当你找到容器所在网址时，最重要的就是该网址下面一定要有一个名为“repodata”的目录存在，那就是容器的网址了。

>yum包目录

/var/cache/yum/x86_64/7/base/packages/

>yum包配置文件目录

/etc/yum.repos.d/

>容器查询

yum repolist all

>查看包信息

yum info packageName
